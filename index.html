Information

Welcome to the Temperature Control System. This web-based Human-Machine Interface (HMI) allows you to monitor and adjust parameters related to temperature control. Here's what you can do on this platform:

Status Monitoring

In the Status tab, you can view the current temperature in real-time, along with the last updated time. You will also see if there are any active alarms, alerting you when the temperature goes outside the set boundaries.

Settings Adjustment

In the Settings tab, you can adjust the high and low alarm limits to define acceptable temperature ranges. If the temperature goes beyond these limits, an alarm will be triggered. You can also fine-tune the system's Filter Alpha, which controls the sensitivity of the temperature filter.

Historical Data

The Historical Data tab allows you to analyze past temperature readings. You can select a custom date range to fetch historical data and visualize it in a dynamic graph.

Dark Mode Toggle

You can switch between light and dark modes using the Dark Mode button in the top-right corner, allowing you to choose a theme thatâ€™s more comfortable for you.

How it Works

The system connects to a backend server that continuously monitors the temperature through sensors. Real-time data is fetched and displayed dynamically. You can interact with the system by adjusting parameters, which are immediately applied, and view historical data for trend analysis.
